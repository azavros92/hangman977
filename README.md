# Hangman
Hangman is a classic game in which a player thinks of a word and the other player tries to guess that word within a certain amount of attempts.

This is an implementation of the Hangman game, where the computer thinks of a word and the user tries to guess it.

## Description
This is a Python-based Hangman game that allows users to guess letters in a randomly chosen word. The goal is to implement the classic game while applying **Python fundamentals** like loops, functions, and conditionals.

### **What I Learned**
- Using Python's `random` module to select words.
- Handling user input with validation.
- Writing reusable functions for better code structure.
